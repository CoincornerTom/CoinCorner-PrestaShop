=== Install ===

Sign up for CoinCorner account at https://www.CoinCorner.com

1. Download https://www.coincorner.com/Content/Plugins/Prestashop-CoinCornerCheckout.zip.
2. Log in to your Prestashop admin area and upload the zip via the module installer.
3. Once uploaded, configure the module and add your API credentials found in the Settings area of your CoinCorner merchant account.
4. Once configured click save and you will be ready to start accepting Bitcoin on your site.
